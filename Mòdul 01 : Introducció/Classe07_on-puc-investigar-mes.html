On puc investigar més sobre JavaScript?

Per completar el que ensenyarem a classe sobre JavaScript, podem recorrer a algunes pàgines de referència:

  • Llocs oficials i documentació:
    • MDN Web Docs (Mozilla): la millor referència per a JavaScript, amb exemples i explicacions clares. Potser una mica complicada de llegir per estudiants que s'inicien en JavaScript
    • JavaScript.info: guia pas a pas molt completa i didàctica.
    • W3Schools: per fer proves ràpides, ideal per principiants.
  • Cursos i vídeos:
    • freeCodeCamp: cursos gratuïts amb exercicis pràctics i certificats.
    • Código facilito: cursos gratuïts i de pagament amb exercicis pràctics.
    • Curso COMPLETO de JavaScript DESDE CERO para Principiantes deMoureDev by Brais Moure: molt recomanat per aprendre des de zero en castellà.
  • Plataformes per practicar:
    • CodePen, JSFiddle i JSBin: entorns en línia per provar codi JavaScript fàcilment.
    • LeetCode i CodeWars: per practicar algoritmes i lògica de programació.
  • Comunitats i fòrums:
    • Stack Overflow: resolució de dubtes i preguntes concretes.
    • Dev.to, Hashnode i Reddit (r/javascript): comunitats actives per compartir coneixement.
