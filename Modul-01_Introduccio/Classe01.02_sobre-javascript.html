Sobre JavaScript

JavaScript és un llenguatge de programació que serveix per afegir interactivitat a les pàgines web.

  • Mostrar missatges quan l’usuari fa clic (alertes, notificacions)
  • Canviar contingut d’una pàgina sense recarregar-la
  • Fer que un botó amagui o mostri una secció
  • Validar formularis abans d’enviar-los
  • Crear jocs, aplicacions, gràfics, etc.

JavaScript fa que una web no sigui només una imatge bonica, sinó que respongui al que fa l’usuari. És responsable d'una revolució completa de les webs.

JavaScript es fa servir tant al frontend com al backend. Al frontend, s’executa dins del navegador de l’usuari i permet fer la pàgina interactiva: mostrar o amagar elements, validar formularis, reaccionar a clics, etc. Al backend, gràcies a entorns com Node.js, JavaScript s’utilitza per crear servidors, gestionar bases de dades i processar sol·licituds des del navegador. Això vol dir que pots fer una web completa només amb JavaScript, tant el que veu i toca l’usuari com el que passa al servidor.

És un des llenguatges més utilitzats i valorats en l'enquesta més important feta als programadors feta anualment a Stack Overflow.

A Europa, algunes de les feines millor pagades que fan servir JavaScript són les de programador web, desenvolupador d’aplicacions o enginyer de programari. Aquestes persones creen pàgines web i aplicacions que funcionen tant al navegador com al servidor. Si saben treballar bé amb JavaScript i eines com React o Node.js, poden guanyar més de 70.000 € l’any en llocs com Alemanya o el Regne Unit. És una molt bona sortida si t’agrada programar i vols fer una feina amb futur.
