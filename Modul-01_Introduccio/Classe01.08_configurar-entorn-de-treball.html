Configurar l'entorn de treball

Necessitarem algunes eines bàsiques com:

  • Visual Studio Code (VS Code): Editor de codi lleuger, gratuït i molt extensible. Recomanem instal·lar extensions com Live Server i Prettier entre d'altres.
  • Google Chrome (o navegador modern): Per provar el codi i accedir a les DevTools (F12) per debugar i inspeccionar elements.
  • JSBin:Es pot executar amb qualsevol navegador web.
  • Node.js: Entorn per executar JavaScript fora del navegador. Instal·lar-lo permet fer servir també el gestor de paquets npm.
  • Git i GitHub: Per controlar versions i compartir projectes. Es pot integrar fàcilment amb VS Code.
  • Terminal o consola: Tant si fas servir macOS, Linux o Windows, cal aprendre a utilitzar la línia d'ordres per executar codi o gestionar fitxers.
  • Navegador de fitxers: Organitza els teus projectes en carpetes clares, com per exemple `projectes-js`, i practica una bona estructura de fitxers.
