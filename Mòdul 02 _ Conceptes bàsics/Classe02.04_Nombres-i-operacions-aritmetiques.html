Nombres i operacions aritmètiques en JavaScript

Amb JavaScript podem fer operacions bàsiques com sumar, restar, multiplicar, dividir, etc. Aquí tens alguns exemples:

Suma (+) 

        
            let a = 10;
            let b = 5;
            let resultat = a + b;
            console.log(resultat); // 15

Resta (-) 

        
            let a = 10;
            let b = 3;
            let resultat = a - b;
            console.log(resultat); // 7

Multiplicació (*) 

        
            let a = 4;
            let b = 6;
            let resultat = a * b;
            console.log(resultat); // 24

Divisió (/) 

        
            let a = 20;
            let b = 4;
            let resultat = a / b;
            console.log(resultat); // 5

Mòdul (%) 

        
            let a = 10;
            let b = 3;
            let resultat = a % b;
            console.log(resultat); // 1

Exponenciació (**) 

        
            let base = 2;
            let exponent = 3;
            let resultat = base ** exponent;
            console.log(resultat); // 8

És molt important que recordeu que = és un operador d'assignació i no d'igualtat. Tenim dos operadors d'igualtat que són == d'igualtat i === d'igualtat estricta.

La llibreria Math

JavaScript inclou la llibreria Math, que ofereix moltes funcions útils per treballar amb nombres.

Math.round() – Arrodoniment a l’enter més proper 

        
            let x = 3.7;
            let arrodonit = Math.round(x);
            console.log(arrodonit); // 4

Math.floor() – Arrodoniment a la baixa (enter inferior) 

        
            let x = 3.7;
            let arrodonit = Math.floor(x);
            console.log(arrodonit); // 3

Math.ceil() – Arrodoniment a l’alça (enter superior) 

        
            let x = 3.2;
            let arrodonit = Math.ceil(x);
            console.log(arrodonit); // 4

Math.pow() – Potències 

        
            let resultat = Math.pow(2, 4);
            console.log(resultat); // 16

Math.sqrt() – Arrel quadrada 

        
            let resultat = Math.sqrt(64);
            console.log(resultat); // 8
