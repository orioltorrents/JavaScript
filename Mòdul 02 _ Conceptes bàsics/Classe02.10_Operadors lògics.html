Operadors lògics a JavaScript

Què són els operadors lògics?

Els operadors lògics serveixen per combinar condicions booleans. Ens ajuden a prendre decisions en el codi segons si certes condicions són certes o falses. Els més utilitzats són:

&& (AND - i)

Retorna true només si totes les condicions són certes.

Exemple && amb dues condicions 

        
            console.log (true && true);
            console.log (true && false);
            console.log (false && true);
            console.log (false && false);
        
    
        
            let edat = 28;
            console.log(edat > 18 && edat < 30);
        
    
        
            let edat = 18;
            let téCarnet = true;
            
            if (edat >= 18 && téCarnet) {
            console.log("Pots conduir.");
            } else {
            console.log("No pots conduir.");
            }

Exemple && amb més de dues condicions

JavaScript les avalua d'esquerra a dreta, i s'atura tan bon punt ja pot determinar el resultat: 

        
            true && false && true  // false - Surt false perquè la segona condició ja és false.

|| (OR - o)

Retorna true si almenys una de les condicions és certa.

Exemple || amb dues condicions 

        
            console.log (true || true);
            console.log (true || false);
            console.log (false || true);
            console.log (false || false);
        
    
        
            let téEntrada = false;
            let ésVIP = true;
            
            if (téEntrada || ésVIP) {
            console.log("Pots entrar a l'esdeveniment.");
            } else {
            console.log("No pots entrar.");
            }

Exemple || amb més de dues condicions

JavaScript les avalua d'esquerra a dreta, i s'atura tan bon punt ja pot determinar el resultat: 

        
            false || true || false // true - Surt true perquè la segona condició és true.

! (NOT - no)

Inverteix el valor booleà: si era true, es torna false i viceversa. 

        
            let estàPlovent = false; // Aquí estàs definint una variable que indica si plou o no. Ara mateix no plou (false).
                        
            if (!estàPlovent) { 
            console.log("Pots sortir sense paraigua.");
            
            // Amb !estàPlovent estàs dient: "si no plou". Com que estàPlovent és false, el ! ho converteix en true, així que el codi dins el if s'executa. Aquest missatge "Pots sortir sense paraigua." surt perquè realment no plou.       

            } else {
            console.log("Agafa un paraigua!");
            }

?? (operador de coalescència nula (nullish coalescing operator)

L’operador ?? retorna el primer valor que NO sigui null ni undefined. A diferència de ||, que retorna el primer valor "truthy", ?? només salta el valor si és exactament null o undefined. Amb ||, el 0, "", false, etc. es consideren falsos i es salten. Amb ??, només es salten si són null o undefined. 

        
            let valor1 = 0;
            let valor2 = valor1 || 42;
            console.log(valor2); // 42 ❌ perquè 0 és "falsy"

            let valor3 = valor1 ?? 42;
            console.log(valor3); // 0 ✅ perquè 0 no és null ni undefined
        
    
        
            let nom = null;
            let nomFinal = nom ?? "Desconegut";
            console.log(nomFinal); // "Desconegut"

No es pot barrejar ?? amb && o || sense parèntesis: 

        
            // ❌ Això dona error:
            true || null ?? "text";

            // ✅ Així sí:
            (true || null) ?? "text";

Conclusió

Els operadors lògics són fonamentals per crear condicions complexes en el nostre codi. Amb &&, || i ! podem controlar millor el comportament del nostre programa.
