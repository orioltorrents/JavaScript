Truthy i Falsy a JavaScript

Què són?

En JavaScript, qualsevol valor pot ser avaluat com a cert o fals en un context booleà, com dins d’un if. Això es coneix com a valors truthy (considerats certs) i falsy (considerats falsos).

Exemple bàsic 

        
            if ("Hola") {
                console.log("És truthy!");
            }

Valors falsy

Aquests són els valors que JavaScript considera falsos: 

        
            false
            0
            -0
            0n
            ""
            null
            undefined
            NaN

Exemple de valor falsy 

        
            if (0) {
                console.log("Això no s'executarà");
            } else {
                console.log("0 és un valor falsy");
            }

Valors truthy

Tots els altres valors que no són falsy es consideren truthy: 

        
            "Hola"
            123
            -5
            []
            {}
            function() {}

Exemple de valor truthy 

        
            if ([]) {
                console.log("Un array buit és truthy!");
            }

Comprovar amb Boolean() 

        
            console.log(Boolean("Hola"));   // true
            console.log(Boolean(""));       // false
            console.log(Boolean([]));       // true
            console.log(Boolean(null));     // false

Ús habitual

Els valors truthy i falsy són útils per validar variables o assignar valors per defecte: 

        
            let nomUsuari = input || "Convidat";

Conclusió

Entendre els valors truthy i falsy permet escriure condicions més netes i eficients. Pots aprofitar aquesta lògica implícita en lloc de fer comparacions directes com === true.
