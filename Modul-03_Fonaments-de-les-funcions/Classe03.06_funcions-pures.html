Funcions pures

Les funcions pures no produeixen efectes secundaris.

Un efecte secundari

Presentem un array let edats = [20] amb un sol valor. Li passem una funció que fa variar l'unic valor de l'array. 

        
            let edats = [20]
            function modificador(edats){
                edats[0] = 25;
            }
            console.log(edats);
            modificador(edats);
            console.log(edats);

Fem una copia de l'array perquè la funció sigui pura 

        
            let edats = [20]
            function modificador(edats){
                let copia = [...edats]
                copia[0] = 25;
            }
            console.log(edats);
            modificador(edats);
            console.log(edats);
