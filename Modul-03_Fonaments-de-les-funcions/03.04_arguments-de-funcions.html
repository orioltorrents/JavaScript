Arguments de funcions

Quan definim una funció, podem indicar paràmetres que li permetin rebre valors des de fora. Aquests valors s’anomenen arguments quan la funció s’executa. D’aquesta manera, les funcions esdevenen molt més flexibles i reutilitzables.

Analogia

Quan definim una funció, indiquem uns paràmetres (com si fóssim cambrers posant plats buits a la taula). Quan cridem la funció, li passem arguments, que són els valors que omplen aquells plats. Un argument emplena un paràmetre.

Exemple bàsic 

        
            // Definim la funció amb un paràmetre
            function saluda(nom) {
                console.log("Hola, " + nom + "!");
            }

            // L'executem passant-li un argument
            saluda("Joana"); // Mostra: Hola, Joana!

Més d’un argument 

        
            function suma(a, b) {
                console.log("La suma és " + (a + b));
            }

            suma(5, 3); // Mostra: La suma és 8

Arguments per defecte

Podem assignar un valor per defecte als paràmetres, per si no s’indica cap valor: 

        
            function saluda(nom = "amic") {
                console.log("Hola, " + nom + "!");
            }

            saluda(); // Mostra: Hola, amic!
            saluda("Laia"); // Mostra: Hola, Laia!

Paràmetres vs Arguments

  • Paràmetres: variables que posem entre parèntesis en definir la funció.
  • Arguments: valors que passem quan cridem la funció.

Activitat

Escriu una funció anomenada presentacio que rebi dos arguments: nom i edat. Ha de imprimir:
"Em dic NOM i tinc EDAT anys." 

        
            function presentacio(nom, edat) {
                console.log("Em dic " + nom + " i tinc " + edat + " anys.");
            }

            presentacio("Pau", 16);
