Variables i constants

Quan executem un programa podem guardar informació útil en la memòria de la computadora, aquesta informació és eliminada quan ja no està en un ús o es tanca el programa.

La informació es guarda en espais de la memòria virtual. Aquests espais de memòria poden tenir noms, i se'n diu variables o constants, depenent de si la informació que hi guardem pot canviar o no.

Variables

Una variable és com una capseta on podem guardar un valor, i més endavant canviar-lo si volem.

En JavaScript, per declarar una variable utilitzem la paraula let o var. Utilitzarem let i més endavant ja us ensenyaré la diferència entre let i var. 

        
            // Declarem una variable nom
            let nom; 

            // Assignem el valor "Anna" a la variable nom
            nom = "Anna";

            // Més tard podem canviar el valor
            nom = "Joan";

Constants

Una constant també guarda informació, però el seu valor no es pot canviar un cop s'ha assignat.

En JavaScript, fem servir const per crear una constant. 

        
            // Declarem una constant
            const pi = 3.14;

            // Això donaria error perquè no podem canviar una constant
            // pi = 3.1416; ❌

Bones pràctiques

  • Utilitza let quan el valor pot canviar.
  • Utilitza const quan el valor ha de ser fix.
  • Evita usar var, ja que pot donar problemes en programes grans.
  • Els noms de les variables han de ser clars, sense espais i no poden començar amb números.
