Variables i constants a JavaScript

Per practicar amb les variables i les constants utilitzarem JS Bin. Entrem-hi! i activem el mòdul de JavaScript a l'esquerra i la consola a la dreta.

Variables

Comencem posant declarant la variable nom posant... 

        
            let nom;

o posant... 

        
            var nom;

La diferència entre let i var l'explicarem més endavant. Per ara, només hem declarat la variable nom però no li hem assignat cap valor. Anem a fer-ho utilitzant l'operador = ... 

        
            nom = "Anna";

Com que "nom" és una variable i li hem assignat un cadena de text o string, JS que és de tipatge dinàmic ja sap que tenim un string dins de la variable nom.

Com que és una variable, li podem assignar un altre cadena de text... 

        
            nom = "Oriol";

Li podem assignar un nombre, si voleu... 

        
            nom = 25;

Noteu que heu assignat un valor a una variable i després li heu assignat dos altres valors més però res ha aparegut en pantalla. No apareix res perquè no li hem dit que faci aparèixer res. Anem-li a dir que imprimeixi la variable a la consola... 

        
            let nom;
            nom = "Anna"
            console.log(nom)

Reassignem el valor de la variable i imprimim-la per consola 

        
            let nom;
            nom = "Anna";
            nom = "Oriol";
            console.log(nom)

Reassignem una segona vegada amb un nombre i imprimim-la per consola 

        
            let nom;
            nom = "Anna";
            nom = "Oriol";
            nom = 27;
            console.log(nom)

També podem imprimir cada vegada el valor de la variable abans de reassignar-la per veure com el valor de la variable va canviant. 

        
            let nom;
            nom = "Anna";
            console.log(nom);
            nom = "Oriol"
            console.log(nom);
            nom = 27;
            console.log(nom)

Per saber quin tipus de variable tenim, podem fer servir la funció typeof... 

        
            let nom;
            nom = "Anna";
            typeof(nom)

Si no la posem dins la consola no veurem quin tipus és 

        
            let nom;
            nom = "Anna";
            console.log(typeof(nom))

Ja hem dit que JavaScript és de tipatge dinàmic i ara ho comprovarem 

        
            let nom;
            nom = "Anna";
            console.log(typeof(nom));
            nom = "Oriol"
            console.log(typeof(nom));
            nom = 27;
            console.log(typeof(nom));
            nom = undefined;
            console.log(typeof(nom))

També podem fer una declaració i una assignació en la mateixa linia 

        
            let nom = "Anna";
            console.log(nom)

Recordem que la variable nom no és la mateixa que Nom. Mirem aquest exemple... 

        
            let nom = "Anna";
            let Nom = "Oriol";
            console.log(Nom)

Recordem que els noms de les variables han de ser descriptius en la mesura que sigui possible. A mesura que aneu programant escrivireu noms de variables més bons i més descriptius.

Hi ha noms de variables que són vàlids... 

        
            let _nom = "Anna";
            let $nom = "Oriol";
            let Nom = "Sílvia";
            let NOM = "Montserrat";
            console.log(_nom);
            console.log($nom);
            console.log(Nom);
            console.log(NOM)

Podem iniciar el nom de les variables amb una lletra, un _ o un $.

però en canvi, n'hi ha noms de variables que NO són vàlids perquè comencen amb un nombre, una caràcter no permès, un operador com el signe -, un @ o contenen un espai... 

        
            let 2nom = "Anna";
            let ?nom = "Oriol";
            let -nom = "Oriol";
            let @nom = "Oriol";
            let nom cognoms = "Oriol Torrents";
            console.log(2nom);
            console.log(?nom);
            console.log(-nom);
            console.log(@nom);
            console.log(nom cognoms);

o que NO són vàlids perquè són paraules reservades per un propòsit concret com ara let, var, const o this entre d'altres. 

        
            let var = "Anna";
            let this = "Oriol";
            console.log(var);
            console.log(this);

Constants

Podem fer amb les constants amb la diferència que NO podem reassignar el valor d'una constant... 

        
            const PI = 3.14;
            console.log(PI)

si fem el següent NO ens funcionarà... 

        
            const PI = 3.14;
            PI = 3.1416;
            console.log(PI)
