Condicions a JavaScript

if

En programació, necessitem moltes vegades condicionar que un bloc de codi s'executi si una condició s'ha complert. Per exemple, si un usuari ha iniciat sessió o si un usuari és major d'edat o no, s'executaran uns codis o uns altres. Per aquestes classes d'avaluacionsm tenim els condicionals. Una condició a JS es declara utilitzant la paraula reservada "if" seguida d'uns parèntesis on hi ha una expressió booleana. Si aquesta expressió és certa, s'executarà el codi que hi ha dins les claus. 

        
            if (condició) {
                // codi a executar si la condició és certa
            }

Per posar el codi a executar si la condició és certa, es recomana posar una tabulació.

Si l'expressió boolaena és falsa, no s'executarà el codi que hi ha dins les claus.

Per exemple, si volem comprovar si un usuari és major d'edat, podem fer això: 

        
            let edat = 18;
            if (edat >= 18) {
                console.log("L'usuari és major d'edat");
            }

Si la condició no és certa, podem afegir una altra clàusula amb la paraula reservada "else" per executar un altre codi. 

        
            if (condició) {
                // codi a executar si la condició és certa
            } else {
                // codi a executar si la condició no és certa
            }

Per exemple, si volem comprovar si un usuari és major d'edat, podem fer això: 

        
            let edat = 17;
            if (edat >= 18) {
                console.log("L'usuari és major d'edat");
            } else {
                console.log("L'usuari no és major d'edat");

També podem afegir més condicions amb la paraula reservada "else if" per executar un altre codi si una altra condició és certa. 

        
            if (condició1) {
                // codi a executar si la condició1 és certa
            } else if (condició2) {
                // codi a executar si la condició2 és certa
            } else {
                // codi a executar si cap condició és certa
            }

Podem crear un exemple on hi ha tres condicions: 

        
            let edat = 17;
            if (edat >= 18) {
                console.log("L'usuari és major d'edat");
            } else if ( edat >= 16) {
                console.log("L'usuari té entre 16 i 17 anys");
            } else {
                console.log("L'usuari té menys de 16 anys");
            }

En JavaScript pots posar tants else if com vulguis. Aquesta estructura serveix per comprovar diverses condicions, una darrere l’altra, en ordre. El programa llegeix de dalt a baix: Quan troba una condició que és true, executa aquell bloc i s’atura. Si cap condició és true, pot acabar amb un else final (opcional).

Bones pràctiques i trucs

  1. Cura amb l'ordre de les condicions

    2. S'ha anar amb molta cura amb l'ordre de les condicions. Sempre escriu de més gran a més petit. Si no ho fem així, podem tenir problemes com ara en aquest exemple: 

            
            if (nota >= 5) {
                // Això s’executaria també per un 9!
            } else if (nota >= 9) {
                // Mai s’executaria perquè el primer ja ha entrat
            }

  2. Evita comparacions sense tipus

    3. == compara només valor i === compara valor i tipus 

            
            console.log("5" == 5);   // true ❗ (no comprova tipus)
            console.log("5" === 5);  // false ✅ (comprova tipus també)

  3. Pots simplificar amb intervals 

    4.         
            if (nota >= 0 && nota < 5) {
                console.log("Suspès");
            }

  4. Cura amb el prompt

    5. Tot el que entra pel prompt és text, cal convertir-ho si vols fer comparacions numèriques: 

            
            let edat = Number(prompt("Quants anys tens?"));

  5. També pots fer servir switch (més endavant)

    6. El switch es fa servir quan les condicions són igualtats concretes. EL veurem més endavant. 

            
            let color = "vermell";
            switch (color) {
            case "vermell":
                console.log("Atura’t!");
                break;
            case "verd":
                console.log("Passa!");
                break;
            default:
                console.log("Color desconegut.");
            }

Exercicis

  1. Parell o senar
    Demana a l’usuari un número i digues si és parell o senar. 
                
                let num = Number(prompt("Introdueix un número"));

                if (num % 2 === 0) {
                console.log("És parell");
                } else {
                console.log("És senar");
                }
  2. Comprovador d’edat
    Demana l’edat de l’usuari i mostra un missatge segons l’edat (infant, adolescent o adult). 
                
                let edat = Number(prompt("Quina edat tens?"));

                if (edat < 12) {
                console.log("Ets un infant");
                } else if (edat < 18) {
                console.log("Ets adolescent");
                } else {
                console.log("Ets adult");
                }
  3. Nota final
    Demana una nota i mostra la qualificació corresponent (Suspès, aprovat, Notable o Excel·lent). 
                
                let nota = Number(prompt("Introdueix la teva nota (0-10)"));

                if (nota >= 0 && nota < 5) {
                console.log("Suspès");
                } else if (nota >= 5 && nota < 7) {
                console.log("Aprovat");
                } else if (nota >= 7 && nota < 9) {
                console.log("Notable");
                } else if (nota >= 9 && nota <= 10) {
                console.log("Excel·lent");
                } else {
                console.log("La nota no és vàlida");
                }
  4. Semàfor
    Demana un color (vermell, groc o verd) i mostra la reacció. 
                
                let color = prompt("Introdueix un color de semàfor").toLowerCase();

                if (color === "vermell") {
                console.log("Atura’t");
                } else if (color === "groc") {
                console.log("Vés amb compte");
                } else if (color === "verd") {
                console.log("Pots passar");
                } else {
                console.log("Color desconegut");
                }
  5. Calculadora senzilla
    Demana dos números i una operació, i mostra el resultat. 
                
                let num1 = Number(prompt("Introdueix el primer número"));
                let num2 = Number(prompt("Introdueix el segon número"));
                let operacio = prompt("Introdueix l’operació (+, -, *, /)");

                let resultat;

                if (operacio === "+") {
                resultat = num1 + num2;
                } else if (operacio === "-") {
                resultat = num1 - num2;
                } else if (operacio === "*") {
                resultat = num1 * num2;
                } else if (operacio === "/") {
                if (num2 === 0) {
                    resultat = "No es pot dividir per zero";
                } else {
                    resultat = num1 / num2;
                }
                } else {
                resultat = "Operació no vàlida";
                }

                console.log("Resultat:", resultat);
