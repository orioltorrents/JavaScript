Operadors de comparació a JavaScript

Què són els operadors de comparació?

Els operadors de comparació ens permeten comparar dos valors i obtenir com a resultat un valor booleà: true o false. Són molt útils per fer condicions amb if, while o altres estructures de control.

Principals operadors de comparació

  • == Igualtat (sense tenir en compte el tipus)
  • === Igualtat estricta (compara també el tipus)
  • != Diferent (sense tenir en compte el tipus)
  • !== Diferent estrictament
  • > Major que
  • >= Major o igual que
  • < Menor que
  • <= Menor o igual que

Exemples pràctics amb Numbers 

        
            // Comparació senzilla
            console.log(5 == '5');      // true (compara el valor, no el tipus)
            console.log(5 === '5');     // false (tipus diferents)
            console.log(10 > 7);        // true
            console.log(3 <= 2);        // false
            console.log(true == 1);     // true (true es converteix en 1)
            console.log(false === 0);   // false (tipus diferents)

Exemples pràctics amb variables que porten Numbers 

        
            let edad = 15;
            let resultat = edad >= 18;
            console.log (resultat)

Exemples pràctics amb Strings

En JavaScript, també podem comparar cadenes de text utilitzant operadors com < i >. Aquesta comparació segueix l’ordre lexicogràfic, és a dir, l’ordre alfabètic segons el valor Unicode dels caràcters. Per exemple, "gat" < "gos" retorna true perquè la lletra "a" ve abans que la "o". Aquesta regla és útil per ordenar llistes de paraules o comprovar quin text va primer alfabèticament. 

        
            const paraula1 = prompt("Escriu una paraula:");
            const paraula2 = prompt("Escriu una altra paraula:");
        
            console.log("Resultat de la comparació:");
            console.log(`"${paraula1}" < "${paraula2}" →`, paraula1 < paraula2);
            console.log(`"${paraula1}" > "${paraula2}" →`, paraula1 > paraula2);
            console.log(`"${paraula1}" === "${paraula2}" →`, paraula1 === paraula2);

En JavaScript (i en molts llenguatges), les lletres majúscules tenen un valor Unicode més baix que les minúscules. Això vol dir que, sorprenentment: 

        
            "A" < "a" // → true
            "Animal" < "animal" → true
            "animal" < "animal" → false

Consell

En general, es recomana utilitzar sempre === i !== per evitar errors deguts a la coacció de tipus automàtica. Així t’assegures que no només el valor, sinó també el tipus, sigui el que esperes.
