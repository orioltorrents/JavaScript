Com és el llenguatge tècnicament

JavaScript és un llenguatge de programació de text, interpretat pel navegador (com Chrome o Firefox). Això vol dir que s’escriu en fitxers .js o dins d’un <script> a HTML, i el navegador l’entén línia a línia, sense necessitat de compilar com passa amb altres llenguatges.

  • Interpretat pel navegador: No cal cap programa especial per executar el codi. Quan obres una pàgina web, el navegador (com Chrome o Firefox) ja entén i executa el JavaScript línia a línia.
  • Tipatge dinàmic: No cal dir si una variable és un número, una paraula, etc. El JavaScript ho entén sol segons el que li donis. Per exemple, let edat = 15; ja sap que és un número.
  • Tipatge dèbil: Si combines variables diferents (com un número i una paraula), JavaScript intentarà adaptar-se sol. Per exemple: "Tens " + 15 dona com a resultat "Tens 15".
  • Case-sensitive: Té en compte les majúscules i minúscules. Les variables Nom i nom no són el mateix!
  • Basat en objectes: Moltes estructures a JavaScript (com textos, números, dates...) es poden utilitzar com si fossin caixes amb eines. Per exemple, una cadena com "Hola" té mètodes com .length per saber quantes lletres té.
  • Orientat a esdeveniments: El JavaScript pot "escoltar" el que fa l’usuari (clicar, escriure, moure el ratolí...) i actuar quan això passa.
  • Multiplataforma: Funciona a qualsevol ordinador o mòbil, sigui del sistema que sigui (Windows, Mac, Android, iPhone...).
