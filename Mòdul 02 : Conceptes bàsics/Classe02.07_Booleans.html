Booleans a JavaScript

Els valors booleans són un dels tipus de dades més simples però potents de JavaScript. Només poden tenir dos valors: true (cert) o false (fals). S’utilitzen per controlar el flux dels programes, especialment amb condicions i bucles.

Declarar un valor boolean 

        
            let ésMajorEdat = true;
            let téCarnet = false;

No és el mateix la paraula reservada false com a booleà que la cadena de text "false".

Expressions booleanes

Les comparacions com ==, ===, >, <, etc., sempre retornen un valor boolean. 

        
            let edat = 18;
            let ésMajor = edat >= 18; // true
            console.log(ésMajor); // mostra true

En JavaScript, un valor boolean sovint s’utilitza com una “bandera” per indicar si una condició s’ha complert o si un estat està actiu o inactiu. Aquesta tècnica és molt útil per controlar el flux del programa de manera clara i eficient.

Una bandera (flag) és una variable que guarda true o false per indicar si una acció s’ha realitzat, si un error ha passat, si una sessió està activa, etc.

Mirem quatre exemples 

        
            let notificacions_actives = true;
        
    
        
            let errorDetectat = false;

            // Simulem una comprovació
            
            let edat = 15;
            
            if (edat < 18) {
            errorDetectat = true;

            if (errorDetectat) {
            console.log("Error: l'usuari no és major d'edat.");

            let sessioIniciada = false;

            function iniciarSessio(usuari, contrasenya) {
                if (usuari === "admin" && contrasenya === "1234") {
                    sessioIniciada = true;
                }
            }

            iniciarSessio("admin", "1234");

En aquest últim codi li podem posar dos prompts per demanar l'usuari i la contrasenya 

        
            // Credencials correctes (simulades)
            const USUARI_CORRECTE = "admin";
            const CONTRASENYA_CORRECTA = "1234";

            // Demanem l'usuari i la contrasenya
            let usuari = prompt("Escriu el teu nom d'usuari:");
            let contrasenya = prompt("Escriu la teva contrasenya:");

            // Bandera per saber si la sessió s'ha iniciat
            let sessioIniciada = false;

            // Comprovem si les credencials coincideixen
            if (usuari === USUARI_CORRECTE && contrasenya === CONTRASENYA_CORRECTA) {
            sessioIniciada = true;
            }

            // Mostrem el resultat
            if (sessioIniciada) {
            console.log("✅ Benvingut, " + usuari + "! Has iniciat sessió correctament.");
            } else {
            console.log("❌ Error: usuari o contrasenya incorrectes.");
            }

Operadors lògics

  • && (AND): retorna true si ambdues condicions són certes
  • || (OR): retorna true si almenys una condició és certa
  • ! (NOT): inverteix el valor boolean
        
            let téCarnet = true;
            let téMultes = false;

            let potConduir = téCarnet && !téMultes; // true && true → true
            console.log(potConduir); // mostra true

Truthiness i Falsiness

En JavaScript, alguns valors no booleans es poden avaluar com true o false en contextos condicionals.

  • Falsos: 0, "", null, undefined, NaN, false
  • Certs: Tots els altres valors
        
            if ("") {
            console.log("Això no es mostrarà");
            } else {
            console.log("És fals!");
            }

Conclusió

Els booleans són essencials per prendre decisions en el codi. Saber com funcionen els valors veritables i falsos, i com utilitzar operadors lògics, és fonamental per escriure condicions eficients i llegibles.
